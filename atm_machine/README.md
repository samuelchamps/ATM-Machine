# ATM Machine Project

This is a beginner Python ATM Machine project.

The program allows users to:
- Enter a PIN
- Check account balance
- Deposit money
- Withdraw money
- Exit the ATM system

---

# Features

- PIN validation
- Balance checking
- Deposit functionality
- Withdrawal functionality
- Insufficient funds check
- Menu system using loops

---

# Concepts Used

This project helped me practice:

- Variables
- If/Else statements
- While loops
- Input handling
- Operators
- Functions
- Conditional logic
